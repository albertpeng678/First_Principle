Phase 1.5 Gate — 三態 error / warn / ok（含 loading + simulation override）

Phase 1 表單提交後，AI 對四個欄位（C1 / I / R / C2 等步驟對應欄位）做框架審核，回 overallStatus + items[]。 任一 error item = 必擋（drill / simulation 一致），唯一動作「返回修改」；warn / ok 通行。三裝置並排：mobile 360 / tablet 768 / desktop 1280。

OK 通行 · drill + simulation
綠色 transition + 全 ok items；可直接點「繼續到 Phase 2」。overallStatus = "ok"
WARN 通行附提醒
綠色 transition（仍通過）+ 1-2 個 warn item 提示可優化點；繼續按鈕仍可用。overallStatus = "warn"
ERROR 阻擋 · drill + simulation 一致
紅色 transition + 阻擋 inline 繼續；底部 sticky bar 唯一動作「返回修改」。任一紅 item = 必擋，無風險繼續路徑。overallStatus = "error"
