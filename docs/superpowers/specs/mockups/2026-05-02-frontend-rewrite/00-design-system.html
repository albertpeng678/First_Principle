Signature Navy（active surface 專用）

--c-navy#1B2D5C
navy-2 hover#142347
navy-lt8% α

導入 navy 取代「黑色 / warm-brown ink」當作 active surface（active pill / brand mark icon / nav tab active / progress active）。Navy 比黑色暖、比 primary 沉穩，與紙感背景配出「正式工作本」氣質。

色票分工原則：
· 中性紙感色（bg / ink / rule）佔畫面 95% — 主導視覺氣氛
· Navy --c-navy #1B2D5C — active surface only：active pill / brand mark icon / nav active / progress active
· Primary --c-primary #1A56DB — action only：primary button / focus ring (3px) / 選中卡片 1px 邊
· 狀態色 4 軌（success / warn / danger / nsm）只在該狀態出現，永不疊加
不准：紫色漸層、彩虹背景、emoji 圖示、把 primary 當底色（保留藍只給「點擊」訊號）
