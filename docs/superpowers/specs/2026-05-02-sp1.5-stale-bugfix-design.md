## 3. Design Rules

### 3.1 Desktop layout split — 頂部 chrome 全寬 / body 居中（D-1）

**Before (production at SP1.5 ship):**
```
.phase2-desktop {
  max-width: 920px;   /* 整個 wrap 居中 */
  margin: 0 auto;
}
```
全部子元素（navbar / circles-nav / progress / chat-body / submit-bar）都被壓到 920 居中。

**After (SP1.5-bugfix):**
```
.circles-chat-wrap.phase2-desktop {
  max-width: none;    /* wrap 全寬 */
}
.circles-chat-wrap .circles-body-centered {
  max-width: 920px;
  margin: 0 auto;     /* 中段 body 居中 */
}
```

頂部 chrome（navbar / circles-nav / progress / banner / qchip）unwrapped → 全寬。
中段 body（chat-body / submit-bar / score-wrap / form）wrapped in `.circles-body-centered` → 920 居中。

**Apply to:** phase 1（renderCirclesPhase1） / phase 2（renderCirclesPhase2） / phase 3（renderCirclesStepScore） desktop。

### 3.2 Stale mode visual — 連續流動，1px ruler 區隔（F-1 P1）

合併 locked-banner + stale-banner 為單條 `.stale-locked-bar`（~44px 高，原本 ~120px）。

```html
<div class="stale-locked-bar">
  <i class="ph ph-warning-octagon icon-warn"></i>
  <div class="core">此題目已更新 — 顯示為唯讀</div>
  <div class="pill"><i class="ph ph-lock-key"></i> 65 分</div>  <!-- only when locked -->
</div>
```

stale 模式下 progress / banner / qchip / chat-body / q-card 統統 edge-to-edge stripe（border-radius: 0），section 之間用 `1px solid var(--c-border)` ruler。

### 3.3 Stale phase 2 / 3 sticky double bar（F-1 UX, P3）

取消 inline phase-back row + 獨立 sticky 回首頁 兩個動作區，合併成一條 sticky bar：

```html
<div class="stale-action-bar">
  <button class="circles-btn-secondary" id="circles-stale-prev">
    <i class="ph ph-arrow-left"></i> 上一步
  </button>
  <button class="circles-btn-primary" id="circles-stale-home">
    回首頁
  </button>
</div>
```

flex 比例 `1 : 1.6`。`#circles-stale-prev` handler：
- Phase 2 stale → 設 `circlesPhase = 1`，回 phase 1 read-only（驗證 user 期望）
- Phase 3 stale → 設 `circlesPhase = 2`，回 phase 2 read-only chat

`#circles-stale-home` handler 統一：clear all CIRCLES state → `navigate('circles')`（F-3 修正）。

### 3.4 Phase 3 stale 按鈕 id（F-2）

Production `app.js:4367` stale 分支用 `id="circles-score-home"`，被 `style.css:4541` 的 navbar icon-btn round-btn rule 命中（`border-radius:50%; background: var(--c-surface)`）。

修正：改用 `id="circles-stale-home"`（與 phase 1/2 stale 統一）。同時加 `#circles-stale-home` click handler 在 `bindCirclesStepScore`。

### 3.5 Stale mode 抑制 drill-encourage（F-2 補）

`app.js:4293` `buildDrillCompleteEncourageHtml()` 在 `AppState.circlesStale === true` 時 return ''。理由：drill mode encouragement card 含「再練一次」按鈕，stale 下 click 會打到當前題庫，與 snapshot 不一致。

### 3.6 Scrollable region clear sticky bar（F-1 P3'）

stale 模式下，所有可 scroll 區域（chat-body / q-card expand / score breakdown）容器加 `padding-bottom: 88px`（覆蓋 sticky bar 高度 12+44+12+16 safe-area）。

### 3.7 Navigate target（F-3）

Audit `tests/audit/MASTER-TEST-PLAN.md:1294` 規定「NO `navigate('home')` anywhere in app.js」。SP1.5 違規：

- `app.js:3188` phase 1 stale-home → 改 `navigate('circles')`
- `app.js:3950` phase 2 stale-home → 改 `navigate('circles')`
- 新加 phase 3 stale-home handler → 用 `navigate('circles')`

### 5.4 `public/app.js` 共用 helper `renderStaleLockedBar`

```js
// SP1.5-bugfix — render compact merged banner (replaces renderLockedBanner + renderStaleBanner in stale mode)
function renderStaleLockedBar(stepKey, stepScores) {
  if (!AppState.circlesStale) return '';
  var locked = isStepLocked(stepKey, stepScores);
  var pillHtml = locked
    ? '<div class="pill"><i class="ph ph-lock-key"></i> ' + Math.round(stepScores[stepKey].totalScore || 0) + ' 分</div>'
    : '';
  return '<div class="stale-locked-bar">' +
    '<i class="ph ph-warning-octagon icon-warn"></i>' +
    '<div class="core">此題目已更新 — 顯示為唯讀</div>' +
    pillHtml +
  '</div>';
}
```

### 5.5 `public/app.js` 共用 handler bind

每個 phase 的 bind 函式（bindCirclesPhase1 / bindCirclesPhase2 / bindCirclesStepScore）加：

```js
document.getElementById('circles-stale-prev')?.addEventListener('click', function() {
  if (AppState.circlesPhase === 2) { AppState.circlesPhase = 1; navigate('circles'); }
  else if (AppState.circlesPhase === 3) { AppState.circlesPhase = 2; navigate('circles'); }
  else { /* phase 1 — no prev */ }
});
document.getElementById('circles-stale-home')?.addEventListener('click', function() {
  AppState.circlesStale = false;
  AppState.circlesSelectedQuestion = null;
  AppState.circlesSession = null;
  AppState.circlesConversation = [];
  AppState.circlesFrameworkDraft = {};
  AppState.circlesStepScores = {};
  AppState.circlesScoreResult = null;
  navigate('circles');
});
```

刪除舊的 phase 1 / phase 2 stale-home handler（line 3180 / 3942）— 統一新 handler。
