**Conclusion**: keep this spec **in the suite** (it tests the F-CT1.2 fix), but reclass it to "robustness / error-injection tier". Not eligible for the real-OpenAI critical-path 5.

### Spec 5 — `tests/e2e/bug3-spinner-deep-investigation.spec.js`

**Purpose** (per header lines 1-30): P2-#253 bug-fix verification — proves `tryResumeLatestSession` now derives `circlesScoreResult` from `step_scores`, so `go-phase3` **skips** `evaluate-step` entirely when the score is already present.

**Conclusion**: keep this spec **in the suite** (it tests the P2-#253 fix), but reclass to "regression/fix-verification tier". Not eligible for the real-OpenAI critical-path 5.
