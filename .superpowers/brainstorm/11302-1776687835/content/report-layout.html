A — Tab 分頁（推薦）

頂部固定總分條 + 4 個 Tab：評分總覽、練習回顧、亮點摘要、匯出。每個 Tab 獨立滾動，手機一欄完整適應。回首頁按鈕在 Navbar。

B — Accordion 折疊區塊

每個區塊可展開/收合，頁面保持緊湊。缺點是使用者需要多一個展開動作才看到內容，對「一次看完報告」的情境不夠流暢。

C — 側邊導覽（桌機）

桌機：左側固定垂直導覽，右側內容區滾動。手機：自動退化成頂部 Tab。最接近 dashboard 工具感，但實作需要兩套 responsive 邏輯。

三種方案共同點
  • Navbar 都有「回首頁」按鈕
  • 頂部固定顯示總分摘要
  • 匯出（PDF/PNG）收進 Navbar 或獨立 Tab，不散落在頁面底部
  • 「再練一次」明確入口
