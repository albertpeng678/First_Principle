C — Heroicons

Tailwind 官方出品，只有約 300 個 icons，需 inline SVG（無 font CDN）。選擇少，實作麻煩，但風格極度精煉。適合已用 Tailwind 的專案。

推薦：Phosphor Icons

此專案純 Vanilla JS + HTML，無打包工具。Phosphor 有官方 font CDN，一行引入即可用 <i class="ph ph-sun">。1300+ icons 涵蓋所有需要的圖示，且 weight 選項可以在不同情境用 regular / bold 做視覺強弱區分。
